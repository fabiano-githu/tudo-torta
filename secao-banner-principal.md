# 🎯 Seção Banner Principal

Implementação do banner principal (hero section) da landing page com foco em **impacto visual**, **chamada para ação (CTA)** e **conversão do usuário**.

---

## 🚀 Visão Geral

O banner principal é a primeira área visual da aplicação, responsável por apresentar o negócio e direcionar o usuário para ações importantes, como pedidos ou navegação.

Foi desenvolvido com **HTML, CSS e Bootstrap**, priorizando responsividade e experiência do usuário.

---

## ⚙️ Funcionalidades

- Exibição de título principal com destaque  
- Subtítulo com mensagem estratégica  
- Botões de ação (CTA), como:
  - Comprar agora  
  - Ver produtos  
- Integração com WhatsApp  
- Layout responsivo  
- Alinhamento centralizado de conteúdo  
- Possibilidade de imagem ou background personalizado  

---

## 🎨 Interface e Design

- Layout moderno e limpo  
- Tipografia com destaque (títulos grandes e legíveis)  
- Cores chamativas para conversão  
- Botões com efeitos de hover  
- Elementos centralizados vertical e horizontalmente  
- Uso de ícones para reforçar ações  

---

## 📱 Responsividade

- Adaptação para diferentes tamanhos de tela  
- Ajustes de fonte para mobile  
- Botões redimensionáveis  
- Imagem responsiva
