# tudo-torta
Minha landing page em html boostrap e javascript.

## 🎨 Interface Base da Landing Page

Foi desenvolvida uma estrutura reutilizável de interface com foco em responsividade, experiência do usuário (UX) e padrão visual profissional.

### ✅ Navbar
- Layout responsivo com Bootstrap
- Hover com animação suave (movimento + mudança de cor)
- Efeito underline animado
- Estrutura pronta para reutilização em múltiplas páginas




### ✅ Footer
- Layout organizado em colunas
- Seção de parceiros com ícones (Bootstrap Icons)
- Hover interativo nos elementos
- Ano atualizado automaticamente via JavaScript
- Estrutura reutilizável em toda aplicação

### ✅ Botão WhatsApp Flutuante
- Posicionamento fixo no canto inferior direito
- Animação pulsante para chamar atenção
- Responsivo para diferentes tamanhos de tela
- Link direto para contato

### ✅ Botão "Voltar ao Topo"
- Aparece apenas ao rolar a página
- Scroll suave até o topo
- Posicionado no canto inferior esquerdo
- Evita conflito visual com o botão do WhatsApp

### ✅ Responsividade
- Ajustes via media queries
- Componentes adaptáveis para mobile
- Espaçamentos e tamanhos otimizados

### ✅ Experiência do Usuário (UX)
- Microinterações com hover e animações
- Feedback visual em botões
- Navegação intuitiva
- Interface limpa e moderna

---

## 🚀 Objetivo

Criar uma base sólida e reutilizável para landing pages, permitindo escalabilidade e padronização visual em todas as seções do projeto.
