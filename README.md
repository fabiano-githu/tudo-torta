## 🚀 Banner Responsivo com Carousel

### 📌 O que foi feito
Implementação de um banner principal utilizando Bootstrap Carousel com foco em responsividade, performance e conversão.

### 🎯 Funcionalidades
- Slider com 3 imagens de fundo
- Layout 100% responsivo (mobile-first)
- Efeito glass moderno (blur + transparência)
- Overlay escuro para melhor legibilidade
- Botões com integração ao WhatsApp
- Grid Bootstrap bem estruturado

### 📱 Responsividade
- Desktop: imagens em `cover` (preenchimento total)
- Mobile: imagens em `contain` (sem cortes)
- Ajustes de altura para melhor visualização

### 🔧 Tecnologias
- HTML5
- CSS3
- Bootstrap 5

### 💡 Impacto
Melhoria significativa na experiência do usuário e aumento de conversão (CTA direto para WhatsApp).
